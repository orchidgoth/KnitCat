# KnitCat 🧶🐱 (+🇩🇪 German version below)

A knitting counter iPhone app for patterns with repeated motifs, increases or decreases. Built with SwiftUI.
 - ‼️You'll need Xcode and/or an iPhone to see it in action.

## Features
- Main counter for knitting rows
- Sub counters for motif and increment/decrement tracking
- Reset button
- Haptic feedback on button press
- ...more features after testing and upon request

## Tech Stack
- Swift
- SwiftUI
- Xcode

## What I'm learning:

- Basics of SwiftUI layout (VStack, ZStack, HStack)
- State management with @State
- Button actions and user interaction
- Git and GitHub workflow
- Translating real-life challenges into a software solution
- Creating an enjoyable user experience
- View changes based on user interactions & data
- Exploring simple software architecture concepts such as separating UI views, state, and logic
